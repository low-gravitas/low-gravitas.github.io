Low Gravitas is a small studio focused on warm, low-blue-light tools for developers — themes, fonts, and utilities that make long coding sessions a little easier on the eyes.

This blog is where we write about the design decisions, technical details, and philosophy behind our projects.

## Projects

- **[Zen Theme](/zen-theme/)** — A warm color theme for VS Code, IntelliJ, Ghostty, iTerm2, Warp, and the web. Dark and light variants with WCAG AA contrast.
- **[Symbol Font](/symbol-font/)** — 11,800+ glyphs from Nerd Fonts, Font Awesome, Material Design, Codicons, and more — in a single font file.
